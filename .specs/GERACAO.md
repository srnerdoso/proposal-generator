# Formulário de Geração

Especifica como o formulário de geração de propostas deve funcionar.

## Entrada

1. [Cliente](#cliente-projeto)
2. [Projeto](#cliente-projeto)

## Saída

1. [Apresentação](#apresentação)
2. [Especificações](#especificações)
3. Oferta

## Cliente Projeto

Dados esperados:

- Nome do cliente
- Título do projeto
- Descrição do projeto

## Apresentação

Deve utilizar os dados persistidos do usuário para gerar uma apresentação profissional e otimizada para o freela com ajuda do agente de IA.

## Especificações

Deve utilizar os dados fornecidos do cliente e projeto para gerar especificações otimizadas.

## Oferta

Deve aferir quantos dias levará até o serviço estar concluído levando em consideração: complexidade, tempo médio de planejamento e tempo médio programando + 2 dias de margem de erro.

## Regras gerais

- Toda a proposta deve ser editável.
